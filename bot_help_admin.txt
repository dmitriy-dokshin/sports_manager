/new - Новая игра, чтобы начать формировать новый список игроков
/schedule <cron> - Установить расписание создания новых игр (время в UTC). Например, "/schedule 30 18 * * 2" - игра по вторникам в 21:30 по Москве.
/delete_schedule - Удалить расписание создания новых игр
/list - Вывести список
/list_aloud - Вывести список с явным упоминанием людей
/random_teams - Сгенерировать две случайные команды
/minus @ (можно указать нескольких) - удалить кого-то
